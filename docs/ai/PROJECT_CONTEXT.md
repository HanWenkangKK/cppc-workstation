# 项目上下文

## 项目名称
CPPC 神经康复 AI 智能工作站（一期 MVP）

## 项目本质
这是一个“医疗垂直 AI 系统”，核心是基于华西 CPPC 知识库生成评估报告和康复方案。

## 核心流程
治疗师勾选/录入评估项 -> 后端组装上下文 -> 调用 AI 工作流(Coze) -> 返回结构化数据 -> 治疗师在线修改 -> 确认归档。

## 当前阶段（一期闭环 - 4周）
只做：
- 患者信息管理（极简）
- CPPC 评估树展示与勾选录入
- 对接 Coze 接口生成报告/方案
- 结构化数据的存储、读取与局部修改保存
- 最终确认并模拟 PDF 导出逻辑

不做：
- 复杂的权限系统（默认单机构/写死用户）
- 多机构租户切换
- 历史版本大重构

## 成功标准
- 治疗师能在一分钟内完成勾选并看到 AI 生成的结构化方案。
- 修改方案中的任意一段文字，刷新后能保持修改后的状态。
